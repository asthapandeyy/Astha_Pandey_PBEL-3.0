# 🎓 AI-Driven Student Performance Prediction System

## 📌 Overview

An AI-powered machine learning application that predicts student performance based on academic factors like study hours, attendance, assignments, and previous scores.

The system provides personalized feedback, performance analysis, and improvement recommendations through an interactive Streamlit dashboard.

---

## 🚀 Features

- Student performance prediction using Machine Learning
- Personalized AI feedback and recommendations
- Performance score calculation
- Academic risk analysis
- Prediction confidence score
- Feature importance visualization
- Confusion matrix evaluation
- Downloadable student performance report

---

## 🛠️ Tech Stack

- **Language:** Python
- **ML Algorithm:** Decision Tree Classifier
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
- **Frontend:** Streamlit

---

## 🤖 Model Details

**Algorithm:** Decision Tree Classifier

**Dataset Size:** 500 student records

**Input Features:**
- Hours Studied
- Attendance
- Assignments
- Previous Score

**Model Accuracy:** 90%

---

## ⚙️ How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run Source_Code/app.py
```
